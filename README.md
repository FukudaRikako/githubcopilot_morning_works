# 🌅 GitHub Copilot Morning Works

**「想像すると同時に、創造する時代へ。」**

朝の時間だけで、GitHub Copilot を使って 6 つのアウトプットを生み出す ── そんな体験を紹介するワークショップ用ランディングページです。

## 📖 概要

このプロジェクトは、GitHub Copilot の活用方法を**非エンジニアにもわかりやすく**伝えるためのワークショップ教材です。朝起きてから家を出るまでの間に Copilot と対話しながら成果物を作り上げる、というストーリー仕立てで構成されています。

## 🎯 対象者

- マーケター・クリエイター
- DX 推進担当者
- 企業の経営企画・事業開発
- GitHub Copilot に興味があるすべての方（プログラミング経験不問）

## 🗂 ファイル構成

```
├── index.html                          # メインページ
├── githubcopilot_morning_works.html    # メインページ（同内容）
└── README.md                           # このファイル
```

## 🚀 使い方

ブラウザで HTML ファイルを直接開くだけで動作します。

```bash
# ローカルで開く
start index.html

# または簡易サーバーで配信
npx serve .
```

## 📋 ページ構成

| セクション | 内容 |
|---|---|
| **ゲームヘッダー** | Copilot マスコットが障害物を飛び越えるミニゲーム |
| **ヒーロー** | 朝の生活動線とアウトプットを重ねたストーリー |
| **6 つのユースケース** | ストーリーページ / プレゼン資料 / ダッシュボード / 3D 可視化 / Web 公開 / AI ナレッジベース |
| **Getting Started** | VS Code インストール → Copilot 開始 → サインインの 3 ステップ |
| **ワークショップ** | 約 30 分のハンズオンプログラム（タイムライン付き） |

## 🛠 技術スタック

- **HTML5 / CSS3 / Vanilla JavaScript**（フレームワーク不使用）
- Canvas API（ミニゲーム）
- Intersection Observer API（スクロールアニメーション）
- Google Fonts（Noto Sans JP）・Mona Sans（GitHub デザインシステム）

## 🎨 デザイン

Copilot ブランドカラーに準拠したグラデーション（Blue `#2563eb` → Purple `#7c3aed`）を基調とし、スクロール連動のフェードイン・タイピングアニメーション・パーティクルエフェクトなどのインタラクションを搭載しています。

## 📝 ライセンス

社内ワークショップ教材としてご利用ください。
