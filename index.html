#6 — Knowledge for AI

つくったものが、AIの知識になる

GitHubに置いたコンテンツは、公開範囲を自分で選べます。
公開すればAIの知識源に。非公開なら社内ナレッジとして安全に活用できます。

🌐
Public（公開）
外に発信したいとき
採用ブランディング、事例紹介、プロダクトページなど、見てほしいコンテンツを世界に公開。
✓ 検索エンジンにインデックスされる
✓ LLMがナレッジとして参照
✓ URLを共有するだけで誰でも閲覧
🔒
Private（非公開）
社内にとどめたいとき
戦略企画、顧客データ、社内プロトタイプなど、守りたい情報は社内だけで管理。
✓ 外部からアクセス不可
✓ Copilotが社外に送信しない
✓ 組織内でSSO・アクセス制御
公開した場合、なぜAIの知識になるのか？ ▼
📖
HTMLは「AIが読める文章」
HTMLはテキストがそのまま構造化されています。
見出し、段落、リストなど、AIが意味を理解しやすい形式です。
🌍
GitHubは世界最大のコード公開プラットフォーム
GitHubの公開リポジトリは、検索エンジンやLLM（大規模言語モデル）の
トレーニングデータとして参照されています。
🔗
URLがあれば、AIが「引用元」として参照できる
Web上に公開されたページはLLMのRAG（検索拡張生成）の情報源に。
あなたの企画がAIの回答の「参照元」として引用される可能性があります。
🔍
検索で見つかる
GoogleやBingの検索結果に表示され、社外にも届く
🤖
AIに引用される
ChatGPTやCopilotが回答の参照元として活用する可能性
♻️
資産として残る
更新が続く限り、鮮度の高いナレッジとして蓄積される

Web上のHTMLは、世界に開かれたナレッジになる。
企画書を書く行為が、そのまま知識資産をつくる行為になります。

企画をつくるだけで、AIの知識が育つ

Getting Started

朝ごはんを食べながら、さぁ準備🍳

3つの手順だけで開始する

1

VS Code をインストール

Visual Studio Code の最新バージョンをダウンロード。GitHub Copilot の拡張機能が統合されています。

code.visualstudio.com →
2

無料で Copilot の使用を開始

無料でのご利用も可能。クレジットカード不要、GitHub アカウントのみ。

※ 無料版は一部の機能でアクセスが制限されます。

3

GitHub アカウントでサインイン

GitHub アカウントで VS Code にサインインすれば、すぐに Copilot が使えるようになります。

🏢 企業・チームでの本格導入をお考えの方へ
Azure Metered（従量課金）プランなら、既存の Azure サブスクリプションで
フル機能の GitHub Copilot をチーム全体に展開できます。
※ 企業向け利用は Microsoft 担当者へ個別にお問い合わせください。

Workshop Program

Snack Time ワークショップ🍪

朝は忙しい？時間がない？大丈夫。わたしたちといっしょに、お菓子を食べながら
GitHub Copilot の設定や使い方を学びましょう。たった30分で完了します。

オープニング & アイスブレイク
ワークショップの目的と進め方を共有。参加者同士の自己紹介タイム。

GitHub Copilotとは？ — デモ & インスピレーション
Copilotの基本機能をライブデモで紹介。企画・マーケティング視点での活用事例を共有します。コード以外にもできることの広さに驚くはず。
00:08 – 00:22
ハンズオン：Copilotで企画をつくってみよう
